A customization framework that enhances site performance and appearance, with support for major plugins.

== Description ==

Comrade offers deep customization options for the look and feel of your site.
In the background, your site's markup and behavior are cleaned and optimized
for speed, search engines, social media, and overall user experience.

Comrade is and always will be free, so you gain access to premium features
without the pricetag or the bloat. Comrade is for the people.

Looking for a helping hand? View plugin documentation (http://websemiotics.com/comrade/documentation/).

== Installation ==

Installing "Comrade" can be done either by searching for "Comrade" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the settings screen and configure, as desired.
